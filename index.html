auto (自动)

auto

浏览器根据当前上下文决定要显示的光标样式。

cursor: auto;

default (默认)

default

通常呈现为箭头，表示默认光标状态。

cursor: default;

pointer (指针)

pointer

表示链接的可点击性，通常呈现为手形图标。

cursor: pointer;

wait (等待)

wait

表示程序正忙，需要用户等待。

cursor: wait;

text (文本)

text

表示文本可被选择，通常呈现为I形光标。

cursor: text;

move (移动)

move

表示某物可被移动。

cursor: move;

not-allowed (禁止)

not-allowed

表示请求的操作将不会被执行。

cursor: not-allowed;

help (帮助)

help

表示有帮助信息可用。

cursor: help;

crosshair (十字线)

crosshair

表示可以精确定位，常用于绘图应用。

cursor: crosshair;
